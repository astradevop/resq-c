NAME:-RESQ

DESCRIPTION:-RESQ is a web application that helps people find nearby emergency services in case of an emergency.

FEATURES:-

SO THREE LOGINS:- 1. CITIZEN 2. VOLUNTEER 3. ADMIN

GLOBAL FEUTURES:-
TAILWIND CSS /FULLY OPTIMISED MOBILE VIEW .
ALL THE SOS AND INCIDENTS SHOULD BE VAILABLE FOR EVERY USERS MAP VIEW 
SHOULD BE REALTIME ,ABSOLUTELY NO NEED TO REFRESH ,EVERYTHING MUST APPEAR ON CORRECTLY WITHOUT DELAY OR REFRESH REQUIRED ,FOR INCIDENTS ,SOS ,CHAT ,ASSIGNMENT ,BROADCAST ETC


CITIZEN:- 
1. REGISTER/LOGIN
2. CURRENT LOCATION AS SOON AS LOGGED IN 
3. SOS BUTTON (WHEN CLICKED IT WILL SEND A REQUEST TO THE ADMIN ,WITH CURRENT LOACTION AND CONTACT DETAILS AND PROFILE DETAILS OF THE CITIZEN .ADMIN SHOULD BE ABLE TO IDENTIFY THE NEAREST VOLUNTEER AND ASSIGN IT TO THEM )
4. INCIDENT REPORT OPTION :-SENT VARIOUS INCDENT REPORT TO THE ADMIN WITH AN OPTION OF CURRENT LOCATION AND SPECIFIC LOCATION TOUCHED OR TYPED AND THEN SELECTED ,WITH DETAILS OF THE INCIDENT .ADMIN SHOULD BE ABLE TO IDENTIFY THE NEAREST VOLUNTEER AND ASSIGN IT TO THEM )
5. A CHAT OPTION WHICH ENABLE CITIZEN TO CHAT WITH THE VOLUNTEER WHO HAS BEEN ASSIGNED TO THEIR SOS OR INCIDENT REPORT  .ALSO TO VIEW BROADCAST FROM ADMIN
6. IMAGE UPLOAD OPTION FOR INCIDENT REPORT (OPTIONAL FOR SURE)

ADMIN:-
1. LOGIN (ADMIN LOGIN IS PRECODED TO PROJECT)
2. MAP VIEW WITH EVERY INCIDENTS AND SOS PROFILES 
3. ADD CRUD FOR VOLUNTEERS,USERS,INCIENTS AND SOS 
4. ASSIGNING TASKS(SOS OR INCIDENT REPORT) TO VOLUNTEERS AND TRACKING 
5. CHAT WITH VOLUNTEERS 
6. BROADCAST TO ALL USERS
7. ADD INFINITE COMMENTS TO TASKS 
8. COMPLETE CONTROL 



VOLUNTEER:-
1. LOGIN
3. DASHBOARD WITH MAP VIEW SHOWING ASSIGNED SOS/INCIDENTS AND NEARBY REPORTS
4. TASK MANAGEMENT: ACCEPT/REJECT ASSIGNMENTS AND UPDATE STATUS (RESPONDING, ON-SITE, COMPLETED)
5. CHAT OPTION: COMMUNICATE WITH THE ASSIGNED CITIZEN AND THE ADMIN
6. VIEW BROADCASTS FROM ADMIN
7. ADD COMMENTS/NOTES TO ASSIGNED TASKS
8. AVAILABILITY TOGGLE (ONLINE/OFFLINE STATUS)
9. GET DIRECTIONS TO SOS AND INCIDENT REPORTS ,NAVIGATION
